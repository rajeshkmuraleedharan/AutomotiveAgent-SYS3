# ASPICE SYS.3 — Work Products

## WP-SYS3-001: System Architectural Design

Content requirements:
- System structure (components and their responsibilities)
- Interfaces between components
- Allocated requirements (which SYS.2 requirement maps to which element)
- Rationale for allocation decisions

## WP-SYS3-002: System Architectural Design Evaluation

- Review record with identified issues
- Traceability verification
- Consistency check between SYS.2 and SYS.3

## WP-SYS3-003: Allocated Requirements

- Complete bidirectional trace matrix
- Each SYS.2 requirement → one or more SYS.3 allocated requirements
- Each SYS.3 requirement → one or more SWE.1 requirements (after SWE.1 baseline)
