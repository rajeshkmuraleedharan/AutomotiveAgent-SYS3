# Confluence — Page Conventions

Conventions the agent follows when reading exported pages or drafting new ones.

## Reading

- H1 = page title; the first paragraph usually carries space/author/date metadata
- Decision pages: look for `Problem / Options / Decision / Consequences` sections —
  these map directly to ADR files in `project-context/decisions/`
- Tables of requirements on pages are informative copies — Codebeamer is authoritative

## Drafting

- One H1, sentence-case title
- Sections H2, sub-sections H3 — no deeper
- Decisions use the `Problem / Options considered / Decision / Consequences / Follow-ups`
  structure (see `imports/samples/confluence-page.html` for the shape)
- Requirements quoted on pages always carry their `SYS-*` ID and a "master in CB" note
