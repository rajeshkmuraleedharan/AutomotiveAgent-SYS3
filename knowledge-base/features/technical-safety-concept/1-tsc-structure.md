# Technical Safety Concept — Structure

## What is the TSC?

The Technical Safety Concept (TSC) is an ISO 26262-4 artefact that translates functional safety requirements (FSR) into technical safety requirements (TSR) allocated to system elements.

## TSC Position in Safety Lifecycle

```
HARA → Safety Goals → Functional Safety Concept (FSC) → FSR
                                                           ↓
                                            Technical Safety Concept (TSC) ← THIS
                                                           ↓
                                               Technical Safety Requirements (TSR)
                                                           ↓
                                                    SWE.1 / Hardware Design
```

## TSC Contents

1. Safety goals (from HARA)
2. FSR → TSR mapping and derivation
3. ASIL decomposition claims
4. Safety mechanism specification
5. FMEDA evidence reference
6. Diagnostic coverage (DC) per mechanism
7. Safe state definition
